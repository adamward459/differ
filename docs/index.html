Differ

A lightweight, native-feeling Git diff viewer built with Electron, React 19, and Tailwind CSS 4.

Open any local repository, browse changed files, and review diffs side-by-side — with inline commenting.

Features

Side-by-side diff

Aligned left/right columns with synchronized scrolling, just like GitHub.

Live file watching

Diffs refresh automatically when you save files, switch branches, commit, or stash.

Inline comments

Click any line to add review comments. Threads are marked outdated when code changes.

Copy comments

One-click export of all active comments to clipboard.

OS-adaptive theme

Light and dark modes via prefers-color-scheme, using OKLCH colors.

Keyboard navigation

Arrow keys, ⌘/Ctrl+Enter to submit, Escape to close.

Tech Stack

Electron 33 React 19 Tailwind CSS 4 Vite 6 electron-vite 5 simple-git Vitest 3 electron-builder

Architecture

Differ uses Electron's main/renderer split with a secure preload bridge. The renderer is a React 19 SPA compiled with the React Compiler babel plugin. Git operations run in the main process via simple-git, and file system changes are watched and debounced before notifying the renderer over IPC.

IPC channels

Minimal, focused channels: folder picker, file list, diff fetching, repo watching, and login-item settings.

Diff parsing

Unified diff output is converted into aligned left/right line arrays for row-aligned side-by-side rendering.

Comment threads

In-memory per session. Threads snapshot line content at creation and are flagged outdated when code changes.
